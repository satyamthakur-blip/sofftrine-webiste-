# Documents Folder

## Company Profile PDF

Please place your company profile PDF in this folder with the filename:
**`company-profile.pdf`**

The website's "Download Company Profile" button links to:
`/documents/company-profile.pdf`

### Instructions:
1. Save your PDF file as `company-profile.pdf`
2. Copy it to this folder: `D:\Coding\Website\dwijainfra-clone\documents\`
3. The download button on the Featured Projects section will automatically work

### File Requirements:
- Filename: `company-profile.pdf` (exactly)
- Format: PDF
- Recommended size: Under 10MB for fast downloads
- Content: Company overview, services, credentials, past projects, contact info
