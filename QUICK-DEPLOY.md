## âš ï¸ TROUBLESHOOTING

### Problem: "404 - File not found"
**Solution:** 
- Wait 5 more minutes
- Check GitHub Pages is enabled (Settings â†’ Pages)
- Verify index.html is uploaded

### Problem: "Images not showing"
**Solution:**
- Did you run `fix-paths.ps1`?
- Check images folder is uploaded
- Clear browser cache (Ctrl + Shift + R)

### Problem: "Website looks broken"
**Solution:**
- Check css folder is uploaded
- Verify paths are relative (no leading /)
- Clear browser cache

### Problem: "Can't see my changes"
**Solution:**
- Wait 2-3 minutes after commit
- Clear browser cache
- Try incognito mode
- Check commit was successful
